Deux types de breadcrumbs, celui qui utilise l'autogénération basé sur le Router d'Angular et le manuel ou on peut spécifier les breads via l'input.

Pour utiliser l'autogénération, activer la propriété [withRouter]="true"

Pour référencer manuellement les breadcrumbs, utiliser le input [breadcrumbs]="YOUR_BREADCRUMBS". L'overflow via les ... est ajouté lorsqu'on 5 niveaux ou plus.
