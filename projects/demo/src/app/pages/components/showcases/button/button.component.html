Les boutons indiquent les actions qui peuvent être exécutées sur une page. Le libellé d'un bouton annonce l'action qui se produit lorsque l'utilisateur interagit avec celui-ci. Pour plus d'information, voir le du Système de design gouvernemental.

Angular Material offre la possibilité de contrôler la couleur via la propriété "color" tout en respectant le thème de l'application.
