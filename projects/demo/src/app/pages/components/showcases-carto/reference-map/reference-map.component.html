La carte de référence correspond à une vignette cartographique permettant la localisation ou la mise en contexte d'un sujet. Elle est placée dans une page informationnelle. Voir la du Système de design gouvernemental.

Elle est optimisée pour une navigation fluide sur tous les appareils. Le zoom et le déplacement sont désactivés lors du défilement et des interactions à un doigt, pour une expérience utilisateur sans interruption.

Pour une navigation fluide, certaines interactions sont bloquées et un message d'aide apparait indiquant l'interaction à réaliser. Le message s'affiche pour une durée de 2 secondes.

  1. Avec la souris, le zoom est bloqué lors du défilement;
  2. Sur appareil avec écran tactile, le déplacement est bloqué au défilement à un doigt.

Configuration de la carte

Il est possible de modifier certaines options globalement comme les libellés ou la durée de l'affichage des messages d'aide. Par exemple, pour les libellés vous devez définir la clé de traduction qui sera utilisée pour récupérer les traductions via le TranslationService
