Le lien en bloc est un élément de navigation qui permet aux utilisateurs de naviguer rapidement vers différentes sections d'une application ou vers un site web externe. Le lien en bloc contient le titre de la section à consulter ainsi que des informations complémentaires. Il peut aussi contenir des liens vers des sous-sections.

Malgré que ce type de composant est utilisé au travers des sites gouvernementaux, aucune spécification officielle n'est disponible pour le moment.

Dans sa forme de base, le lien en bloc contient un titre, une description, ainsi qu'un lien menant à la section ou un site web voulu:

@for (section of sections1; track section) { }

Le lien en bloc peut aussi contenir des liens vers des sous-sections ou des sites web. La description est facultative lorsque des liens sont présents. Le titre du lien en bloc est plus foncé lorsque la description est absente:

@for (section of sections2; track section) { }

Finalement, le lien en bloc peut être accompagné d'une icône. L'icône est plus foncée lorsqu'il n'y a pas de description:

@for (section of sections3; track section) { }
