Le menu d'ancres est un élément de navigation interne qui permet aux utilisateurs de naviguer rapidement vers différentes sections d'une page Web. Il est particulièrement utile pour les longues pages où le repérage de l'information peut être laborieux. Pour plus d'information, voir le du Système de design gouvernemental.

Chaque lien pointe vers un titre de niveau 2 (h2) dans la page. Les autres niveaux de titre sont omis du menu d'ancres.
